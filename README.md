# 💣 扫雷游戏

经典扫雷游戏，纯前端实现，支持电脑和手机端游玩。

## 在线体验

[👉 立即开始](https://ltwza.github.io/minesweeper/)

## 功能特性

### 🎮 核心玩法
- 标准扫雷规则，左键揭格、右键插旗
- 双击数字格子自动揭开周围（chord 操作）
- 四种预设难度 + 自定义模式

| 模式 | 棋盘 | 雷数 | 限时 |
|:----|:----|:----|:----|
| 😊 简单 | 9×9 | 10 | — |
| 🤔 中等 | 16×16 | 40 | — |
| 😈 困难 | 24×20 | 99 | — |
| 🔥 地狱 | 28×30 | 160 | — |
| 🎨 自定义 | 自由设置 | 自由设置 | 可选 |

### 💾 存档系统
- **自动存档**：游戏退出时自动保存进度
- **手动存档**：最多 9 个手动存档位
- 支持读取、覆盖、删除、重命名
- slot 0 始终是自动存档，不会被删除

### 📊 排行榜 & 通关记录
- **排行榜**：按难度拆分 5 个榜单，简单/中等/困难/地狱按用时排序，自定义按时间倒序
- **通关记录**：混合显示所有已通关记录，最新在前，最多 10 条

### 🎵 背景音乐
- 3 首内置 MP3 背景音
- 支持上传本地音乐文件作为自定义背景音
- 独立调节音效和背景音量

### 📱 移动端适配
- 响应式棋盘布局
- 长按弹出插旗菜单（点开 / 插旗模式）
- 支持触摸操作

### 🏠 本地存储
所有数据保存在浏览器本地，无需登录：
- 游戏设置（音量、音轨选择）
- 存档进度
- 排行榜和通关记录
- 自定义模式配置
- 自定义音乐文件

## 操作说明

| 操作 | 电脑端 | 手机端 |
|:----|:-------|:-------|
| 揭开格子 | 左键点击 | 点击 / 长按 |
| 插旗/取消旗 | 右键点击 | 游戏中长按 → 拖动到"插旗"按钮 或 长按松手 |
| 批量揭雷 | 双击数字 | 双击数字 |
| 插旗模式 | — | 游戏中长按 → 拖动到"⛳ 插旗"按钮进入，再次长按选择"✋ 点开"退出 |

## 本地开发

```bash
# 克隆仓库
git clone https://github.com/ltwza/minesweeper.git

# 直接用浏览器打开 index.html 即可运行
# 推荐用 Live Server 等本地服务器预览
```

项目为纯前端单页应用，无构建步骤，无依赖包。

## 技术栈

- 原生 HTML / CSS / JavaScript
- localStorage（设置、存档、记录持久化）
- IndexedDB（自定义音乐文件存储）
- Web Audio API（音频解码与循环播放）
- CSS Grid（棋盘布局）
- 响应式设计
- GitHub Pages 部署
